\begin{ex}
Let $(X, d)$ be a metric space and $(xn)$ be a sequence in $X$. Then $xn to x$ if and only if for every open set $U$ in $X$ containing $x$, there exists $n0 in naturals$ such that $xn in U for all n greater than n0$.
\end{ex}
